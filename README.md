# Rahamotullah — Portfolio

A personal portfolio site for Rahamotullah, Mechanical Engineer (CAD / R&D), built with
plain HTML, CSS and JavaScript — no build tools required.

## View it live

Once deployed with GitHub Pages (see steps below), your site will be at:

```
https://<your-github-username>.github.io/<repo-name>/
```

## Files

- `index.html` — page content
- `style.css` — all styling
- `script.js` — small interactive bits (mobile nav, scroll reveal)
- `assets/` — profile photo, project images, and downloadable CV PDF (keep this folder
  name and structure — the page references files as `assets/profile.jpg`,
  `assets/project-rov.jpg`, `assets/Rahamotullah_Manik_CV.pdf`, etc.)

## Editing

Everything is in plain HTML/CSS, so you can open `index.html` and `style.css` in any
text editor and change text, links, or colors directly — no rebuild step needed.

## Deploying with GitHub Pages

1. Create a new repository on GitHub (e.g. `portfolio`).
2. Upload `index.html`, `style.css`, `script.js`, and the whole `assets` folder (drag the
   folder itself into the GitHub upload box — it will keep the `assets/` path automatically).
3. Go to the repo's **Settings → Pages**.
4. Under "Build and deployment", set **Source** to `Deploy from a branch`, branch
   `main`, folder `/ (root)`. Click **Save**.
5. Wait a minute, then visit `https://<your-github-username>.github.io/<repo-name>/`.
